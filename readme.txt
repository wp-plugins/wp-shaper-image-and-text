== Description ==
WP Shaper Image and Text is a dynamic image & text widget plugin for display sidebar or any where in your site. 

The plugin is designed for any dynamic image & text widget. "WP Shaper Image and Text" is a free wordpress plugin. Any developer can change Or modify this plugin without author permission.




== Installation ==
1. Upload `wpshaper-image-and-text` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widget area & active "WP Shaper Image and Text widget" very easy, where you want to dispaly dynamic image & text.
